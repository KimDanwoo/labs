# Awesome Saju - PRD (Product Requirements Document)

## 1. 프로젝트 개요

**목표**: 다른 사주 플랫폼과 차별화되는 **완벽한 정확도**의 사주 계산 엔진 구축

**핵심 가치**: 분(分) 단위 절기 정밀 계산, 학파별 옵션 지원, 엣지 케이스 완전 처리

**기술 스택**: Next.js 16 + TypeScript + FSD 2.1

---

## 2. Phase 1 — 사주 계산 엔진 (Core Engine)

UI보다 계산 정확도가 우선. 엔진을 먼저 구축하고 검증한 뒤 화면을 구성한다.

### 2-1. 입력

| 항목      | 필수 | 설명                                 |
| --------- | ---- | ------------------------------------ |
| 생년월일  | O    | 양력 기준 (음력 입력 시 양력 변환)   |
| 생시      | O    | 정확한 시각 (HH:mm) → 시진 자동 변환 |
| 성별      | O    | 대운 순행/역행 결정에 필수           |
| 음력 여부 | △    | 음력 입력 시 양력 변환 후 계산       |

**지원 범위**: 1900년 ~ 2100년 (절기 DB 커버 범위)

### 2-2. 사주팔자 도출 (Four Pillars)

#### 년주(年柱)

- 천간: `(year - 4) % 10`
- 지지: `(year - 4) % 12`
- **경계 기준: 입춘(立春) 입기 시각** (양력 1월 1일도, 음력 정월 초하루도 아님)
- 입춘 이전 출생자 → 전년도 간지 적용

#### 월주(月柱)

- **12절(節) 기준**으로 월 경계 판단 (중기 아님!)
- 절기 입기 시각 이전 출생 → 이전 월주 적용

| 절기 | 황경 | 월지 | 절기 | 황경 | 월지 |
| ---- | ---- | ---- | ---- | ---- | ---- |
| 입춘 | 315° | 寅   | 입추 | 135° | 申   |
| 경칩 | 345° | 卯   | 백로 | 165° | 酉   |
| 청명 | 15°  | 辰   | 한로 | 195° | 戌   |
| 입하 | 45°  | 巳   | 입동 | 225° | 亥   |
| 망종 | 75°  | 午   | 대설 | 255° | 子   |
| 소서 | 105° | 未   | 소한 | 285° | 丑   |

- 월간은 **년간 기준 인월(寅月) 천간표**에서 도출:

| 년간  | 인월 천간 |
| ----- | --------- |
| 甲/己 | 丙寅      |
| 乙/庚 | 戊寅      |
| 丙/辛 | 庚寅      |
| 丁/壬 | 壬寅      |
| 戊/癸 | 甲寅      |

#### 일주(日柱)

- JDN(Julian Day Number) 기반 계산
- 기준: 1900년 1월 1일 = JDN 2415021 = 甲戌일 (60갑자 인덱스 10)
- `일주 인덱스 = (JDN - 2415021 + 10) % 60`
- **반드시 복수의 알려진 날짜로 교차검증**

#### 시주(時柱)

- 12시진(時辰) 매핑:

| 시진 | 시간        | 시진 | 시간        |
| ---- | ----------- | ---- | ----------- |
| 子   | 23:30~01:30 | 午   | 11:30~13:30 |
| 丑   | 01:30~03:30 | 未   | 13:30~15:30 |
| 寅   | 03:30~05:30 | 申   | 15:30~17:30 |
| 卯   | 05:30~07:30 | 酉   | 17:30~19:30 |
| 辰   | 07:30~09:30 | 戌   | 19:30~21:30 |
| 巳   | 09:30~11:30 | 亥   | 21:30~23:30 |

- 시간 → 일간 기준 자시(子時) 천간표에서 도출:

| 일간  | 자시 천간 |
| ----- | --------- |
| 甲/己 | 甲子      |
| 乙/庚 | 丙子      |
| 丙/辛 | 戊子      |
| 丁/壬 | 庚子      |
| 戊/癸 | 壬子      |

### 2-3. 절기(節氣) 정밀 계산

**가장 중요한 모듈** — 절기 시각의 오차가 곧 사주 오차

- Jean Meeus "Astronomical Algorithms" 공식 기반 태양 황경 계산
- 이진탐색으로 목표 황경 통과 시각 산출 (±5분 이내)
- 1900~2100년 × 24절기 = **4,800개 레코드 사전 계산 → DB화**
- 절기 당일 출생자는 DB에서 입기 시각(분 단위)과 비교

### 2-4. 정확도 핵심 이슈 — 사용자 옵션으로 제공

| 이슈               | 설명                                           | 처리 방식                        |
| ------------------ | ---------------------------------------------- | -------------------------------- |
| **야자시(夜子時)** | 23:30~00:00 일주 기준 학파 차이                | 조자시(다음날)/야자시(당일) 옵션 |
| **태양시 보정**    | 표준시(135°E) vs 실제 경도(127°E) = ~32분 차이 | 태양시 적용 ON/OFF 옵션          |
| **역사적 시간대**  | 1961년 이전 UTC+8:30 적용                      | 출생 연도별 자동 보정            |
| **대운수 나머지**  | 일수÷3 나머지 처리 학파 차이                   | 버림/반올림 옵션                 |

### 2-5. 한국 표준시 이력 (자동 적용)

| 기간      | UTC 오프셋     |
| --------- | -------------- |
| ~1908     | +8:27:52 (LMT) |
| 1908~1912 | +8:30          |
| 1912~1945 | +9:00 (JST)    |
| 1945~1954 | +9:00          |
| 1954~1961 | +8:30          |
| 1961~     | +9:00 (KST)    |

## 3. Phase 1-B — 분석 모듈

사주팔자 도출 후, 아래 분석을 순차적으로 구현한다.

### 3-1. 오행(五行) 분석

- 천간/지지 각각의 오행 판별 (木火土金水)
- 음양 구분 (陽: 甲丙戊庚壬 / 陰: 乙丁己辛癸)
- 사주 내 오행 분포 계산 (과다/부족 판별)

### 3-2. 십신(十神)

일간 기준 나머지 7글자와의 관계:

| 관계        | 같은 음양  | 다른 음양  |
| ----------- | ---------- | ---------- |
| 동일 오행   | 비견(比肩) | 겁재(劫財) |
| 내가 생(生) | 식신(食神) | 상관(傷官) |
| 내가 극(剋) | 편재(偏財) | 정재(正財) |
| 나를 극(剋) | 편관(偏官) | 정관(正官) |
| 나를 생(生) | 편인(偏印) | 정인(正印) |

- 지지의 십신은 **지장간 정기(正氣)** 기준

### 3-3. 지장간(地藏干)

| 지지 | 여기   | 중기  | 정기   |
| ---- | ------ | ----- | ------ |
| 子   | 壬(10) | -     | 癸(20) |
| 丑   | 癸(9)  | 辛(3) | 己(18) |
| 寅   | 戊(7)  | 丙(7) | 甲(16) |
| 卯   | 甲(10) | -     | 乙(20) |
| 辰   | 乙(9)  | 癸(3) | 戊(18) |
| 巳   | 戊(7)  | 庚(7) | 丙(16) |
| 午   | 丙(10) | 己(9) | 丁(11) |
| 未   | 丁(9)  | 乙(3) | 己(18) |
| 申   | 戊(7)  | 壬(7) | 庚(16) |
| 酉   | 庚(10) | -     | 辛(20) |
| 戌   | 辛(9)  | 丁(3) | 戊(18) |
| 亥   | 戊(7)  | 甲(7) | 壬(16) |

### 3-4. 12운성(十二運星)

- 양간 순행, 음간 역행
- 12단계: 장생→목욕→관대→건록→제왕→쇠→병→사→묘→절→태→양
- 각 천간의 장생지:

| 천간  | 장생지 | 방향 |
| ----- | ------ | ---- |
| 甲    | 亥     | 순행 |
| 乙    | 午     | 역행 |
| 丙/戊 | 寅     | 순행 |
| 丁/己 | 酉     | 역행 |
| 庚    | 巳     | 순행 |
| 辛    | 子     | 역행 |
| 壬    | 申     | 순행 |
| 癸    | 卯     | 역행 |

### 3-5. 합충형파해(合沖刑破害)

**천간합**: 甲己(土), 乙庚(金), 丙辛(水), 丁壬(木), 戊癸(火)

**지지 육합**: 子丑(土), 寅亥(木), 卯戌(火), 辰酉(金), 巳申(水), 午未(火)

**삼합**: 寅午戌(火), 亥卯未(木), 申子辰(水), 巳酉丑(金)

**방합**: 寅卯辰(木), 巳午未(火), 申酉戌(金), 亥子丑(水)

**육충**: 子午, 丑未, 寅申, 卯酉, 辰戌, 巳亥

**형**: 寅巳申(지세), 丑戌未(무은), 子卯(무례), 辰辰/午午/酉酉/亥亥(자형)

**파**: 子酉, 丑辰, 寅亥, 卯午, 巳申, 未戌

**해**: 子未, 丑午, 寅巳, 卯辰, 申亥, 酉戌

### 3-6. 공망(空亡)

| 순(旬) | 범위      | 공망  |
| ------ | --------- | ----- |
| 甲子순 | 甲子~癸酉 | 戌·亥 |
| 甲戌순 | 甲戌~癸未 | 申·酉 |
| 甲申순 | 甲申~癸巳 | 午·未 |
| 甲午순 | 甲午~癸卯 | 辰·巳 |
| 甲辰순 | 甲辰~癸丑 | 寅·卯 |
| 甲寅순 | 甲寅~癸亥 | 子·丑 |

- 일주 기준 공망 계산: `60갑자 인덱스 // 10 * 10` → 해당 순의 공망 지지

### 3-7. 대운(大運) / 세운(歲運)

**대운 방향**:

- 양남(甲丙戊庚壬년 남) + 음녀(乙丁己辛癸년 여) → 순행
- 음남 + 양녀 → 역행

**대운수 계산**:

- 순행: 출생일 → 다음 절(節)까지 일수 ÷ 3 = 대운 시작 나이
- 역행: 이전 절(節) → 출생일까지 일수 ÷ 3 = 대운 시작 나이

**대운 간지**: 월주에서 순행/역행으로 10년마다 1칸씩 이동

**세운**: 해당 연도의 년주 간지

## 4. 데이터 요구사항

### 코드 내장 (정적 룩업 테이블)

- 60갑자 배열
- 천간/지지 오행·음양 매핑
- 지장간 테이블 (36행)
- 합충형파해 매핑 테이블
- 12운성 조견표 (120값)
- 공망 테이블 (6순 × 2)
- 월간/시간 천간 기준표
- 한국 표준시 이력 테이블

### 사전 계산 DB (절기)

- 1900~2100년 × 24절기 = ~4,800 레코드
- 각 레코드: `{ year, termIndex, solarLongitude, kstDatetime }`
- Jean Meeus 공식으로 사전 생성

### 외부 연동 (선택)

- 음양력 변환: KASI API 또는 자체 삭(朔) 계산

## 5. 검증 전략

### 교차 검증 대상

- 한국천문연구원 절기 시각 데이터
- 검증된 만세력 사이트 3곳 이상과 결과 비교
- 알려진 유명인 사주 (공개된 사주팔자와 대조)

### 테스트 케이스

| 케이스                      | 검증 포인트    |
| --------------------------- | -------------- |
| 입춘 당일 출생 (시각 전/후) | 년주/월주 경계 |
| 절기 당일 출생 (매월 12절)  | 월주 경계      |
| 23:30~00:30 출생            | 야자시 처리    |
| 1961년 이전 출생            | 시간대 보정    |
| 1900년 / 2100년 경계        | 범위 한계      |
| 윤달 음력 입력              | 음양력 변환    |

### 자동화 테스트

- 최소 100개 이상의 known-good 사주 결과를 테스트 픽스처로 확보
- 각 모듈별 단위 테스트 + 전체 파이프라인 통합 테스트
- 절기 시각 검증: KASI 공개 데이터와 ±10분 이내 오차 확인

## 7. 구현 순서 (우선순위)

```
Phase 1-A: 사주팔자 도출 (정확도 최우선)
  1. 천간/지지 기초 데이터 + 타입 시스템
  2. JDN 변환 + 일주 계산
  3. 절기 DB 생성 (Jean Meeus 공식)
  4. 년주 계산 (입춘 경계 처리)
  5. 월주 계산 (절 기반 경계 + 월간 도출)
  6. 시주 계산 (시진 매핑 + 야자시 옵션)
  7. 시간대 이력 보정 + 태양시 옵션
  8. ✅ 검증: 100+ 테스트 케이스 통과

Phase 1-B: 분석 모듈
  9. 오행 분석
  10. 십신 계산
  11. 지장간
  12. 12운성
  13. 합충형파해
  14. 공망
  15. 대운/세운
  16. ✅ 검증: 분석 결과 교차 검증

Phase 2: UI (Phase 1 검증 완료 후)
  - 입력 폼
  - 사주 결과 시각화
  - 만세력 뷰
```
