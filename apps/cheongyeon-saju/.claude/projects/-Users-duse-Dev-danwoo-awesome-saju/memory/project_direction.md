사주 풀이를 AI 캐릭터가 웹툰 스타일로 설명하는 컨셉.

**UI 방향**:
- 매력적인 AI 캐릭터 (웹툰 스타일 일러스트) — 에셋 제작 중, 플레이스홀더로 시작
- 캐릭터가 사주 결과를 대화형으로 풀이
- 웹툰처럼 시각적으로 표현

**AI 풀이**:
- Claude API로 사주 계산 결과를 자연어 해석 생성
- 캐릭터 말풍선으로 표현

**수익 모델** (결제 연동은 나중에):
- 기본 사주 결과는 무료로 일부 공개
- 상세 풀이/분석은 블러(가림) 처리
- 결제하면 전체 내용 공개 (paywall)

**화면 구성**:
1. 입력 화면 — 생년월일시 + 성별
2. 결과 화면 — 캐릭터가 웹툰처럼 풀이 (무료/유료 영역 분리)

**Why:** 차별화를 위해 단순 텍스트 결과가 아닌 캐릭터 기반 인터랙티브 경험 제공
**How to apply:** UI 설계 시 캐릭터 중심 레이아웃, 결제 전/후 콘텐츠 분리 구조 필수
